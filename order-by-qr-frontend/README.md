
## 🛠️ Tech Stack

* **Framework:** React (Vite)
* **State Management:** React Hooks (Context API / Redux)
* **Real-time:** `useWebSocket` cho kết nối hai chiều.
* **Styling:** CSS Modules / Tailwind CSS (Tùy chỉnh theo phong cách hệ thống).
* **API Client:** Axios (Tích hợp JWT cho các yêu cầu bảo mật).
