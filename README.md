# 🧾 QR Ordering System

**QR Ordering System** là hệ thống gọi món bằng mã QR dành cho quán ăn.  
Khách hàng chỉ cần **quét mã QR** trên bàn, hệ thống sẽ hiển thị **menu** đúng với bàn đó, cho phép đặt món và gửi đơn hàng trực tiếp đến **nhân viên**
