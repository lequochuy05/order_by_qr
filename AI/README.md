# 🍜 AI Resource Management - Order By QR

This directory contains the necessary scripts to train and deploy the dish classification model using a Cloud-based workflow.

## 🚀 Cloud Workflow (Roboflow + Colab)

The system is designed to handle data and training in the cloud to save local resources and leverage free GPUs.

### Step 1: Manage Data & Labels (Roboflow)
1. Use [Roboflow](https://roboflow.com) to upload and label your food photos.
2. Generate a version and obtain your **Roboflow API Key** and **Project ID**.
3. *Note: You don't need to store images in this local directory.*

### Step 2: Training (Google Colab)
1. Open the [train_classifier.py](file:///home/qhuy/Documents/order_by_qr/AI/scripts/train_classifier.py) script.
2. Ensure your Roboflow credentials are set in the `CONFIG` section.
3. Upload and run this script on **Google Colab** (using a GPU runtime).
4. The script will:
    *   Download the dataset directly from Roboflow to Colab.
    *   Train the YOLOv8-cls model.
    *   Export the result to **TF.js** format.

### Step 3: Deployment (Local Frontend)
Once training is finished in Colab, download the exported model files and place them here:
`frontend/public/models/dish-classifier/`

Required files:
*   `model.json`
*   `group1-shard*of*.bin`
*   `labels.json` (Automatically generated by the script)
